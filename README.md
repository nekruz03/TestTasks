# Группировка строк

## Описание:

Программа группирует строки из файла по совпадению значений в одинаковых колонках

## Примечание

В тестовом файле `lng-4.txt` все значения были в кавычках. Кавычки удалены, чтобы файл соответствовал условию задачи.
Чтобы программа могла обработать данные, кавычки были удалены с помощью команды:

```bash
sed -i '' 's/"//g' lng-4.txt 
```


### Алгоритм:

1. Берём файл и читаем строчку за строчкой
2. Пропускаем плохие строки
3. Если строчка уже была — вторую не берём
4. Собираем в группы — складываем в одну группу те строчки, у которых на одинаковых местах (в одной колонке) есть одинаковые непустые значения
5. Выводим результат — показываем, сколько получилось групп, где больше одной строчки

## Результат
Количество групп с более чем одним элементом: **1910**
## Время выполнение 
Благодаря использованию Union-Find, смог реолизовать программу, которая быстро работает и укладывается в огроничения

Запуск на тестовом файле **lng-4.txt** с ограничением памяти 1 ГБ показал:

* **Время выполнения:** около 2 секунд (1,98 с)
